# Robotics Book

A comprehensive guide to robotics with ROS 2, Gazebo, Isaac, VSLAM, and VLA.

## About This Book

This book provides both theoretical knowledge and practical hands-on experience with modern robotics frameworks. It's designed to take readers from robotics fundamentals to advanced topics with practical implementations.

## What You'll Learn

- Fundamentals of robotics and autonomous systems
- Practical implementation using ROS 2
- Simulation techniques with Gazebo
- Advanced perception with Isaac and VSLAM
- Visual-language-action models with VLA

## Prerequisites

- Basic programming knowledge
- Understanding of linear algebra and calculus
- Familiarity with Linux command line

## Getting Started

1. Install Node.js 18+ and npm
2. Clone this repository
3. Run `npm install` to install dependencies
4. Run `npm start` to start the development server

## Development

To start the development server:

```bash
npm start
```

To build the static site:

```bash
npm run build
```

To serve the built site locally:

```bash
npm run serve
```
