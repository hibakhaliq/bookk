# Chapter 1: Assessment

## Assessment Overview

This assessment evaluates your understanding of fundamental robotics concepts, including robot components, classifications, applications, and basic ROS 2 setup. The assessment includes multiple-choice questions, short answers, and practical exercises.

## Assessment Type
- **Format**: Mixed (Multiple Choice, Short Answer, Practical)
- **Duration**: 90 minutes
- **Passing Score**: 70%

## Multiple Choice Questions

### Question 1
Which of the following is NOT a typical component of a robotic system?
A) Sensors
B) Actuators
C) Controllers
D) Paint Brushes

### Question 2
What does ROS stand for?
A) Robot Operating System
B) Robotic Operating Software
C) Robot Operation Suite
D) Robotic Operation System

### Question 3
Which type of robot is designed to operate in underwater environments?
A) Aerial robot
B) Terrestrial robot
C) Aquatic robot
D) Anthropomorphic robot

### Question 4
What is the primary function of a robot's controller?
A) To sense the environment
B) To move the robot
C) To process information and make decisions
D) To communicate with humans

### Question 5
Which ROS 2 command is used to list active topics?
A) ros2 node list
B) ros2 topic list
C) ros2 list topics
D) ros2 show topics

## Short Answer Questions

### Question 6
Explain the difference between a sensor and an actuator in a robotic system. Provide two examples of each.

### Question 7
Describe three different classifications of robots based on their application areas. For each classification, provide an example and explain its primary function.

### Question 8
What are the main components of a typical mobile robot? Explain the function of each component.

### Question 9
Describe the basic setup process for a ROS 2 development environment. What are the key steps involved?

### Question 10
Explain the concept of robot autonomy. What are the different levels of autonomy, and provide an example for each level.

## Practical Exercise

### Exercise 1: ROS 2 Basic Commands
Perform the following tasks and document your results:
1. Check if ROS 2 is properly installed on your system
2. List all active ROS 2 nodes
3. List all active ROS 2 topics
4. Create a simple publisher that publishes "Hello, Robotics!" every 2 seconds
5. Verify that the publisher is working correctly

### Exercise 2: Robot Component Identification
Given a simple robot diagram (provided), identify and label the following components:
- All sensors
- All actuators
- Controller
- Power system
- Communication interface

## Answer Key

### Multiple Choice Answers:
1. D) Paint Brushes
2. A) Robot Operating System
3. C) Aquatic robot
4. C) To process information and make decisions
5. B) ros2 topic list

### Short Answer Rubric:
- Question 6: 10 points (5 points for each concept with examples)
- Question 7: 15 points (5 points for each classification with examples)
- Question 8: 12 points (3 points for each component with function)
- Question 9: 8 points (2 points for each key step)
- Question 10: 15 points (5 points for concept explanation, 10 points for autonomy levels)

### Practical Exercise Rubric:
- Exercise 1: 20 points (4 points for each task completed correctly)
- Exercise 2: 20 points (4 points for each component correctly identified)
